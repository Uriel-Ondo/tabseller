
# TabSeller

Site de vente de tableaux.

## Description

TabSeller est une plateforme en ligne permettant la vente de tableaux d’art. Le projet combine une interface utilisateur en HTML et une logique backend en Python.

## Fonctionnalités

- Présentation de galeries de tableaux
- Pages produit détaillées
- Panier d’achat
- Gestion des commandes
- Authentification des utilisateurs

## Technologies utilisées

- **HTML** : pour la structure et l’affichage du site.
- **Python** : pour la logique serveur (framework à préciser, par exemple Flask ou Django).

## Installation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/Uriel-Ondo/tabseller.git
   cd tabseller
   ```

2. Installez les dépendances Python (si applicable) :
   ```bash
   pip install -r requirements.txt
   ```

3. Lancez l’application (exemple avec Flask) :
   ```bash
   flask run
   ```
   *(Adaptez cette commande selon votre framework)*
